Dokumentacja N Engine Libraries
Struktura MipMapInfo
Przechowuje informacje dotyczące mip mapy.
Zdefiniowane składowe:
Size
Rozmiar mip mapy w bajtach.
Dimensions
Rozmiar mip mapy - szerokość i wysokość.
PixelsDataPtr
Wskaźnik na piksele mip mapy.
Offset
Przesunięcie początku mipmapy względem początku wczytanego pliku DDS.
