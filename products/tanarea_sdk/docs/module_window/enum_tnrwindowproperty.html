Dokumentacja Tanarea SDK
Enumeracja tnrWindowProperty
Typ enumeracji: Podstawowa
Typ podstawowy: ui8
Zawiera wartości modyfikujące sposób wyświetlania okna.
Zdefiniowane wartości:
VISIBLE
Okno jest widoczne.
FULLSCREEN_MODE
Okno jest wyświetlane w trybie pełnego ekranu.
RESIZABLE
Okno może mieć rozmiar zmieniany w czasie rzeczywistym przez użytkownika.
DECORATED
Okno będzie posiadać systemowy pasek dekoracyjny.
TRANSPARENT_FRAMEBUFFER
Okno będzie posiadać możliwość renderowania zawartości okna jako przezroczyste.
OPENGL_CORE_4_0_SURFACE
Okno będzie powiązane z kontekstem OpenGL w wersji 4.0 Core Profile.
