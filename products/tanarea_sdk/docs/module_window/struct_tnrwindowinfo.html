Dokumentacja Tanarea SDK
Struktura tnrWindowInfo
Przechowuje informacje na temat okna.
Zdefiniowane składowe:
DisplayPtr
Wskaźnik do wyświetlacza reprezentowany przez klasę tnrDisplay. Wymagany inny niż nullptr dla operacji związanych z wyświetlaczem,
takie jak centrowanie okna czy też wyświetlanie w trybie pełnoekranowym.
Position
Pozycja okna. Pole opcjonalne, domyślna wartość to std::nullopt.
Size
Rozmiar okna. Musi być większe niż 0, zarówno dla szerokości jak i wysokości.
RefreshRate
Częstotliwość odświeżania. Domyślna wartość to 0.
Properties
Modyfikatory okna.
WindowTitle
Tytuł okna.
