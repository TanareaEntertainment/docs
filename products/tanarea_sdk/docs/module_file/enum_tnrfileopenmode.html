Enumeracja tnrFileOpenMode
Typ enumeracji: Podstawowa
Typ podstawowy: ui8
Dostarcza zestaw flag, które modyfikują sposób w jaki plik ma zostać otwarty.
Zdefiniowane wartości:
READ
Umożliwia otwieranie plików w trybie do odczytu.
WRITE
Umożliwia otwieranie plików w trybie do zapisu.
ALLOW_TRUNCATE
Otwiera plik usuwając jego całą dotychczasową zawartość.
CREATE_IF_NO_EXIST
Jeśli podczas otwierania pliku okaże się, że plik nie istnieje, flaga ta gwarantuje, że zostanie on automatycznie utworzony.
