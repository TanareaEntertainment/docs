Składowa tnrFileBinary::ReadBinary()
Odczytuje bajty z pliku binarnego i zwraca w postaci typu podanego jako argument szablonu. Zawsze ilość odczytanych bajtów jest równa wielkości podanego typu.
tnrFileBinary::ReadBinary<T>()
T
Określa typ danych jakie mają zostać zwrócone z odczytanego pliku.
Generowane wiadomości
0x00000004
0x00000003
