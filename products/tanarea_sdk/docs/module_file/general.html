Dokumentacja Tanarea SDK
File - Ogólne
Moduł File dostarcza zunifikowany interfejs do zarządzania plikami. Umożliwia odczytywanie, zapisywanie oraz tworzenie plików i więcej.
Moduł Meta inf znajduje się w przestrzeni nazw tnr::file.
Ponadto wymaga zależności:
- tnr_sdk_string
- tnr_sdk_file
- tnr_sdk_debug_layer (Opcjonalnie)
