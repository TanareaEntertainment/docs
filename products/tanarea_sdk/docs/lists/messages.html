Dokumentacja Tanarea SDK
Lista wiadomości
Tutaj znajdziesz informacje dotyczące wiadomości wysyłanych przez różne moduły SDK - kody, opisy.
Wiadomości
0x00000000
Nieznany kod wiadomości. Jeśli widzisz tę wiadomość, zgłoś to deweloperom Tanarea SDK gdyż należy to traktować jako błąd w kodzie SDK.
0x00000001
Otwieranie pliku wymaga zdefiniowanego przynajmniej jednego trybu. Rzuca wyjątkiem tnrInvalidOpenMode.
0x00000002
Plik nie istnieje i flaga CREATE_IF_NO_EXIST nie została zastosowana. Rzuca wyjątkiem tnrFileNotExist.
0x00000003
Dalsze odczytywanie pliku może spowodować niezdefiniowane zachowanie.
0x00000004
Próba odczytu pliku, który nie został otwarty w trybie do odczytu. Rzuca wyjątkiem tnrUnreadableFile.
0x00000005
Próba zapisu do pliku, który nie został otwarty w trybie do zapisu. Rzuca wyjątkiem tnrUnwritableFile.
0x00000006
Okno nie może zostać utworzone z rozmiarem 0x0. Minimalny rozmiar okna to 1x1.
0x00000007
By przełączyć okno w tryb pełnego ekranu wymagane jest uprzednie przypisanie okna do ekranu (można to zrobić metodą SetDisplay).
0x00000008
Nie można wyśrodkować okna na tym wyświetlaczu, ponieważ jego tryby wideo są nieznane.
0x00000009
Nie można wyśrodkować okna na tym wyświetlaczu, ponieważ jego tryby wideo są nieznane.
0x0000000A
Nie można wykonać operacji zapis/odczyt na pamięci, która nie została przez widok pozyskana.
Rzuca wyjątkiem tnrSharedMemoryViewUnallowedIO.
0x0000000B
Ścieżka do plik wykonywalnego nie została określona. Nie można utworzyć procesu bez pliku wykonywalnego.
Rzuca wyjątkiem tnrInvalidExecutablePath.
0x0000000C
Tworzenie nowego procesu się nie powiodło. Rzuca wyjątkiem tnrProcessCreationFailed.
0x0000000D
Tworzenie nowego procesu się nie powiodło.
0x0000000E
Uzyskanie kodu wyjścia procesu się nie powiodło. Rzuca wyjątkiem tnrRetrieveExitCodeFailed.
0x0000000F
Zdefiniowano kilka argumentów. Implementacja dla systemów Windows obsługuje tylko jeden argument,

z tego powodu pozostałe argumenty zostaną pominięte.
0x00000010
Nazwa gniazda nie może mieć długości 0 znaków. Rzuca wyjątkiem tnrInvalidInterprocessSocketAddressName.
0x00000011
Nie można czekać na zewnętrzne prośby o połączenie jako nie-właściciel gniazda domeny.
Obiekt tnrInterprocessSocket musi być utworzony z flagą SOCKET_OWNER. Rzuca wyjątkiem tnrInterprocessSocketUnowned.
0x00000012
Nie można przyjmować zewnętrznych próśb o połączenie z gniazdem domeny jako nie-właściciel gniazda domeny.
Obiekt tnrInterprocessSocket musi być utworzony z flagą SOCKET_OWNER.
0x00000013
Wymagany zestaw funkcji Winsock jest niedostępny. Rzuca wyjątkiem tnrInterprocessSocketWinsockUnsupported.
0x00000014
Nie można uzyskać właścicielstwa nad gniazdem domeny, ponieważ to gniazdo domeny już ma właściciela.
Rzuca wyjątkiem tnrInterprocessSocketUnownable.
0x00000015
Prośba o połączenie z gniazdem domeny się nie powiodła. Rzuca wyjątkiem tnrInterprocessSocketRequestFailed.
0x00000016
Nie można czekać na utworzenie gniazda jako właściciel ów gniazda. Rzuca wyjątkiem tnrCantWaitAsSocketOwner.
0x00000017
Oczekiwane gniazdo nie powstało. Rzuca wyjątkiem tnrInterprocessSocketNoExist.
0x00000018
Żądany rozmiar dla pamięci współdzielonej powinien być większy niż 0. Rzuca wyjątkiem tnrSharedMemorySizeMustBeNotZero.
0x00000019
Alokacja pamięci współdzielonej nie powiodła się. Rzuca wyjątkiem tnrSharedMemoryAllocationFailed.
0x0000001A
Nie udało się dołączyć procesu do pamięci współdzielonej. Rzuca wyjątkiem tnrSharedMemoryJoinFailed.
0x0000001B
Maska dostępu niezdefiniowana. Musi być przynajmniej WRITE/READ. Rzuca wyjątkiem tnrSharedMemoryHandleNoAccess.
0x0000001C
Nie uwolniono eksluzywnego dostępu do pamięci współdzielonej.
0x0000001D
Zapis do pamięci współdzielonej bez uprawnień do zapisu jest niedozwolony. Rzuca wyjątkiem tnrSharedMemoryViewBadAccess.
0x0000001E
Odczyt do pamięci współdzielonej bez uprawnień do odczytu jest niedozwolony. Rzuca wyjątkiem tnrSharedMemoryViewBadAccess.
0x0000001F
Widok wykracza poza maksymalny zasięg bufora. Rzuca wyjątkiem tnrSharedMemoryViewOutOfRange.
0x00000020
Podane dane wykraczają poza zasięg widoku. Dane poza zasięgiem zostaną zignorowane.
0x00000021
Pozyskiwanie już pozyskanego przez ten sam widok eksluzywnego dostępu do pamięci współdzielonej.
Potencjalne zakleszczenie, ponieważ pamięć współdzielona nie może być pozyskana dopóki nie zostanie zwolniona, nawet jeśli pozyskuje ją ten sam widok.
0x00000022
Nieprawidłowy uchwyt do pamięci współdzielonej. Musi to być uchwyt, który już dołączył proces do pamięci współdzielonej, podano natomiast nullptr.
Rzuca wyjątkiem tnrInvalidSharedMemoryHandle.
0x00000023
Zasięg widoku pamięci współdzielonej musi być określony. Podano std::nullopt Rzuca wyjątkiem tnrUnspecifiedSize.
0x00000024
Próba pozyskania już pozyskanego eksluzywnego dostępu do pamięci współdzielonej przez inny widok.
0x00000025
Pamięć współdzielona nie może być zwolniona przez nie-pozyskiwacza.
0x00000026
Próba odczytu/zapisu z/do międzyprocesowego gniazda jako jego właściciel. Rzuca wyjątkiem tnrInterprocessSocketUnvallowedIO.
0x00000027
Tworzenie okna z kontekstem OpenGL 4.0 jest przestarzałe. Zalecane jest użycie OpenGL 4.2 tak często jak to możliwe.
0x00000028
Próba podmiany buforów, które nie istnieją. Utwórz tnrWindow z flagą OPENGL_CORE_4_0_SURFACE lub OPENGL_CORE_4_2_SURFACE.
Rzuca wyjątkiem tnrNoDoubleBuffering.
0x00000029
Żaden kontekst OpenGL nie jest przypisany do okna. Utwórz tnrWindow z flagą OPENGL_CORE_4_0_SURFACE lub OPENGL_CORE_4_2_SURFACE.
Rzuca wyjątkiem tnrInvalidContext.
0x0000002A
Nie można przypisać okna jako instancja Vulkan na oknie przypisanym do kontekstu OpenGL. Rzuca wyjątkiem tnrInvalidWindow.
