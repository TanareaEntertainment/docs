Dokumentacja Tanarea SDK
Enumeracja tnrSDKVersion
Typ enumeracji: Klasowa
Typ podstawowy: ui32
Zawiera wspierane wersje Tanarea SDK przez obecną instalację.
Zdefiniowane wartości:
VERSION_1_0_0
Wersja 1.0.0.
VERSION_1_1_0
Wersja 1.1.0.
INSTALLED_VERSION
Zawsze jest równa najwyższej wersji wspieranej przez zainstalowaną wersję na maszynie kompilującej.
Historia zmian
- v1.0.0: Dodano.
- v1.1.0: Dodano oznaczenie dla wersji 1.1.0.
