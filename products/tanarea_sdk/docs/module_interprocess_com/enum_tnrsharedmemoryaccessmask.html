Dokumentacja Tanarea SDK
Enumeracja tnrSharedMemoryAccessMask
Typ enumeracji: Podstawowa
Typ podstawowy: ui8
Maska bitowa - określa permisje widoku.
Zdefiniowane wartości:
NO_ACCESS
Brak dostępu.
WRITE
Dostęp do zapisu do pamięci współdzielonej.
READ
Dostęp do odczytu z pamięci współdzielonej.
