Składowa tnrSharedMemoryAllocator::Allocate()
Alokuje pamięć współdzieloną i zwraca do niej uchwyt, za pomocą którego dostęp otrzymuje proces alokujący.
Tworzy również obiekt tnrSharedMemoryAllocator i zwraca jego adres. Metoda statyczna.
tnrSharedMemoryAllocator::Allocate(CreationInfo, SharedMemoryHandle)
CreationInfo
Zestaw informacji wymagany do alokacji pamięci współdzielonej.
SharedMemoryHandle
Parametr wyjściowy, za jego pomocą jest zwracany obiekt klasy tnrSharedMemory.
Generowane wiadomości
0x00000018
0x00000019
