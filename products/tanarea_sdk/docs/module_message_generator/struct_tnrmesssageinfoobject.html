Struktura tnrMessageInfoObject
Dziedziczy pola z struktury tnrMessageInfo. Przechowuje rozszerzone informacje dotyczące wiadomości.
Przeznaczona jest do wiadomości dotyczących konkretnych obiektów w pamięci.
Zdefiniowane składowe:
ObjectAddress
Adres obiektu, którego dotyczy wiadomość.
ObjectTypeName
Nazwa typu obiektu, na który wskazuje pole ObjectAddress.
