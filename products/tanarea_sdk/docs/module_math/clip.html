Funkcja clip()
Przycina wartość Value aby mieściła się w granicach Min i Max.
clip(Mix, Max, Value)
Min
Dolna granica, poniżej której Value ma ustaloną wartość Min.
Max
Górna granica, powyżej której Value ma ustaloną wartość Max.
Value
Wartość do przycięcia.
