Klasa mat2
Klasa reprezentująca matematyczną macierz 2x2.

Dostępne są warianty o różnym typie podstawowym:
- i8mat2
- i16mat2
- i32mat2
- i64mat2
- ui8mat2
- ui16mat2
- ui32mat2
- ui64mat2
- f32mat2
- f64mat2

Dodatkowo klasa mat2 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=

Część z tych operatorów jest kompatybilna z klasą vec2. Istnieje tylko jeden wymóg - obie klasy muszą mieć ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.

Układ elementów macierzy w pamięci jest w układzie rzędowym.
Składowe klasy
mat2::data()
mat2::determinant()
mat2::inverse()
mat2::mat2()
mat2::rotate()
mat2::scale()
mat2::transpose()
