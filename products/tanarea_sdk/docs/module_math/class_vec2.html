Klasa vec2
Klasa reprezentująca matematyczny wektor dwuwymiarowy.

Dostępne są warianty o różnym typie podstawowym:
- i8vec2
- i16vec2
- i32vec2
- i64vec2
- ui8vec2
- ui16vec2
- ui32vec2
- ui64vec2
- f16vec2
- f32vec2
- f64vec2

Dodatkowo klasa vec2 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=
- operator==

Wywołanie operatorów jest możliwe tylko, gdy oba wektory mają ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.
Składowe klasy
vec2::abs()
vec2::distance()
vec2::dot()
vec2::length()
vec2::normalize()
vec2::vec2()
vec2::x
vec2::y
Historia zmian
- v1.0.0: Dodano.
- v1.1.0: Dodano wariant z typem podstawowym fp16.
