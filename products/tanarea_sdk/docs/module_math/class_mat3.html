Klasa mat3
Klasa reprezentująca matematyczną macierz 3x3.

Dostępne są warianty o różnym typie podstawowym:
- i8mat3
- i16mat3
- i32mat3
- i64mat3
- ui8mat3
- ui16mat3
- ui32mat3
- ui64mat3
- f32mat3
- f64mat3

Dodatkowo klasa mat3 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=

Część z tych operatorów jest kompatybilna z klasą vec3. Istnieje tylko jeden wymóg - obie klasy muszą mieć ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.

Możliwe jest również rzutowanie klasy mat3 do mat2. Układ elementów macierzy w pamięci jest w układzie rzędowym.
