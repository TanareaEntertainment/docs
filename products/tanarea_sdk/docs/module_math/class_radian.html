Klasa radian
Przechowuje wartość kąta w radianach. Można go rzutować na klasę degree, tym samym konwertując go do stopni.

Dostępne są dwa warianty:
- fp32
- fp64

Można również przechowywaną wartość otrzymać poprzez rzutowanie radian na typ podstawowy. Zalecana jest jednak ostrożność,
gdyż klasa radian służy przede wszystkim uniknięciu pomyłki względem nieodpowiednich jednostek stopni podczas obliczeń.
Składowe klasy
radian::radian()
