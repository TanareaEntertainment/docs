Klasa vec3
Klasa reprezentująca matematyczny wektor dwuwymiarowy.

Dostępne są warianty o różnym typie podstawowym:
- i8vec3
- i16vec3
- i32vec3
- i64vec3
- ui8vec3
- ui16vec3
- ui32vec3
- ui64vec3
- f16vec2
- f32vec3
- f64vec3

Dodatkowo klasa vec3 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=
- operator==

Wywołanie operatorów jest możliwe tylko, gdy oba wektory mają ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.

Możliwe jest również rzutowanie typu obiektu do klasy vec2.
Składowe klasy

vec3::cross()

vec3::dot()
vec3::length()

vec3::vec3()
