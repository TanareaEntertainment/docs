Klasa vec4
Klasa reprezentująca matematyczny wektor dwuwymiarowy.

Dostępne są warianty o różnym typie podstawowym:
- i8vec4
- i16vec4
- i32vec4
- i64vec4
- ui8vec4
- ui16vec4
- ui32vec4
- ui64vec4
- f32vec4
- f64vec4

Dodatkowo klasa vec4 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=
- operator==

Wywołanie operatorów jest możliwe tylko, gdy oba wektory mają ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.

Możliwe jest również rzutowanie typu obiektu do klasy vec2 oraz vec3.
