Klasa mat4
Klasa reprezentująca matematyczną macierz 4x4.

Dostępne są warianty o różnym typie podstawowym:
- i8mat4
- i16mat4
- i32mat4
- i64mat4
- ui8mat4
- ui16mat4
- ui32mat4
- ui64mat4
- f32mat4
- f64mat4

Dodatkowo klasa mat4 posiada zdefiniowane operatory:
- operator[]
- operator+
- operator+=
- operator-
- operator-=
- operator*
- operator*=
- operator/
- operator/=

Część z tych operatorów jest kompatybilna z klasą vec4. Istnieje tylko jeden wymóg - obie klasy muszą mieć ten sam typ podstawowy.
Wspomnianą kompatybilność można zapewnić dzięki możliwości statycznego rzutowania między typami podstawowymi za pomocą standardowego static_cast<>.

Możliwe jest również rzutowanie klasy mat4 do mat2 i mat3. Układ elementów macierzy w pamięci jest w układzie rzędowym.
