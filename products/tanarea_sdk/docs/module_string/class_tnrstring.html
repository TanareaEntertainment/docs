Klasa tnrString
Klasa tnrString dostarcza interfejsu do manipulacji łańcuchami tekstowymi.

Pozwala na konkatenację łańcuchów tekstowych za pomocą operatorów + i +=. Zawarty łańcuch znaków można iterować operatorem [].

Dodatkowo operator== pozwala na porównywanie obiektów tnrString czy przechowują taki sam łańcuch tekstowy.

Klasa posiada również zdefiniowany operator<< - w związku z tym można go używać do przepychania zawartego tekstu do strumieni std::ostream.
