Składowa tnrString::tnrString()
Konstruktor klasy tnrString.

1. Konstruktor kopiujący tekst z innego obiektu tnrString.
2. Konstruuje obiekt z łańcuchem znaków.
3. Konstruktor domyślny, konstruuje obiekt z pustym łańcuchem znaków.
tnrString::tnrString(String)
String
Obiekt klasy tnrString, z którego ma zostać skopiowany tekst.
tnrString::tnrString(Text)
Text
Łańcuch znaków jaki ma zawierać nowo konstruowany obiekt.
