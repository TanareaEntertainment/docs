Składowa tnrString::from_string()
Konwertuje i zwraca tekst w postaci binarnej. Zależnie od wybranego wariantu zwraca dane o różnym typie podstawowym.

- from_string_ui64
- from_string_ui32
- from_string_ui16
- from_string_ui8
- from_string_i64
- from_string_i32
- from_string_i16
- from_string_i8
- from_string_f64
- from_string_f32
