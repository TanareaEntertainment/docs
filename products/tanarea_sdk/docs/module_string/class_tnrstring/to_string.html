Składowa tnrString::to_string()
Metoda statyczna. Konwertuje i zwraca podaną wartość binarną w postaci tekstowej.

Dostępne są przeciążenia dla typów podstawowych:
  • i8
  • i16
  • i32
  • i64
  • ui8
  • ui16
  • ui32
  • ui64
  • f32
  • f64
tnrString::to_string(Value)
Value
Wartość binarna do przekonwertowania.
