Dokumentacja Tanarea SDK
Korzystanie
1. Gdy masz już zainstalowany pakiet Tanarea SDK, możesz rozpocząć korzystanie z niego.
Aby to zrobić dołącz do swojego projektu nagłówek <TanareaSDK.hpp>
2. Jeśli Twój system budowania nie może znaleźć ścieżki do nagłówka <TanareaSDK.hpp> i/lub bibliotek dla linkera, należy je dodać do projektu.
Ścieżka do bibliotek dla linkera znajduje się pod zmienną środowiskową $TNR_SDK_DIR oraz $TNR_SDK_DIR/externals.
Natomiast do pliku nagłówkowego prowadzi ścieżka $TNR_SDK_DIR/include oraz $TNR_SDK_DIR/externals/include.
3. Wszystkie moduły Tanarea SDK działają oddzielnie, w związku z tym wszystkie biblioteki należy dołączać do linkera manualnie z osobna.
4. Jeśli dla jakichś modułów są dostarczone biblioteki dynamiczne dla Twojej platformy, należy je dostarczyć razem z aplikacją (jeśli są przez nią używane).
