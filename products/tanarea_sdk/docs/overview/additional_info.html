Dokumentacja Tanarea SDK
Dodatkowe informacje
1. SDK zostało napisane w standardzie C++23 (wersja 1.0.0 w C++20). Nie posiada również modułów dla żadnego innego języka.
2. Tanarea SDK działa tylko na systemach 64-bitowych. Systemy 32-bitowe nie są wspierane.
3. W przypadku kompilacji samego SDK wymagane są zależności:
- paczka TanareaSDK-Externals
- CMake 3.24 lub nowszy
- Ninja 1.10 lub nowszy
- Kompilator zależnie od platformy (v1.0.0):
- GCC 11.4 (Linux)
- Clang 17.0.5 (Windows)
- Kompilator zależnie od platformy (v1.1.0):
- GCC 13.1 (Linux)
- Clang 18.1.0 (Windows)
