Dokumentacja Tanarea SDK
Instalacja
1. Pobierz instalator ze strony Tanarea Entertainment.
2. Uruchom pobrany instalator (pamiętaj żeby to zrobić z uprawnieniami administratora).
3. Zalecane jest, by pozwolić instalatorowi zainstalować wszystkie wymagane zależności.
W przeciwnym wypadku SDK może działać niepoprawnie.
4. Jeśli chcemy odinstalować Tanarea SDK, w przypadku systemu Windows postępujemy jak z innymi aplikacjami.
Natomiast w przypadku systemu Linux wystarczy usunąć katalog /tnr_sdk
